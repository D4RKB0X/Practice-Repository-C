# My C++ Projects

Welcome to my collection of C++ projects, These projects reflect my journey learning C++ through both fundamental exercises and creative mini-games inspired by Warcraft 3 and other themes. The repository is organized into categories for clarity and skill demonstration.

---

## 📂 Arrays Collection
Projects that focus on array manipulation, searching, and analysis.

- **Array_AverageCalculator.cpp** – Calculate the average of numbers in an array.
- **Array_FindElement.cpp** – Search for a specific element within an array.
- **Array_PriceAnalysis.cpp** – Analyze prices (min, max, average) from a dataset.
